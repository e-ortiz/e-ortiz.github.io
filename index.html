PPM_One Revisited:
PPM_One is used to display 3D Molecules in real time to study.
Originally, PPM_One was much too slow and inefficient to run and use as a tool to study. With some molecules taking up to 11 hours to run, it was a barely used program that could be a lot more useful if improved.
My team and I worked deligintely with the chemistry department to bring PPM_One into a functional order.
My refactoring the code, we were able to kink out any bad software development practices and see a drop from 11 hours to about 5 from that alone.
But we didn't stop there, by going through the code once again and parallilizing it where we saw fit were were able to use OpenMPand the help of some Nvidia GPUs to drastically improve the speed of the program.
Succesfully, we can proudly say that that originally large 11 hour molecule has has it's runtime drastically improve to a mere 10 minutes runtime! A great accomplishment.
Now, PPM_One is being used to help find a cure to HIV.
